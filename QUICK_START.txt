================================================================================
  SETUP INSTRUCTIONS
================================================================================

STEP 1: Install Python Dependencies
  Run in terminal:
  > pip install -r requirements.txt

STEP 2: Setup Database
  Make sure MySQL is running, then:
  > python setup_database.py
  
  This will create:
  - authors table (with 5 sample authors)
  - titles table (with 5 sample titles)
  - titleauthor table (with relationships)

STEP 3: Start the Application
  Option A - Use the start script:
  > start.bat
  
  Option B - Run directly:
  > python web_app.py

STEP 4: Open in Browser
  Navigate to: http://localhost:5000

================================================================================
  FEATURES
================================================================================

AUTHORS TAB:
  - Add authors with full contact info (name, phone, address, city, state, ZIP)
  - Edit existing authors
  - Delete authors (orphaned titles are also deleted)
  - Contract status tracking

TITLES TAB:
  - Add titles with comprehensive details (price, advance, royalty, sales, etc.)
  - Link to multiple authors with individual royalty percentages
  - Edit existing titles
  - Delete titles

VIEW BY AUTHOR TAB:
  - Filter titles by author
  - See royalty percentages
  - View publication dates

================================================================================
  TROUBLESHOOTING
================================================================================

Problem: Database connection error
Solution: 
  - Check MySQL is running
  - Verify BooksDB database exists
  - Check credentials in web_app.py

Problem: Port 5000 already in use
Solution:
  - Change port in web_app.py (last line)
  - Use: app.run(debug=True, host='0.0.0.0', port=5001)

Problem: Module not found
Solution:
  - Run: pip install -r requirements.txt
